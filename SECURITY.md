# Security Policy

## Supported Versions

Use this section to find out which versions of this mod are currently being supported with security updates and critical bug fixes.

| Version           | Supported          |
| ----------------- | ------------------ |
| 1.X.X+26.X.X      | :white_check_mark: |
| < 1.0.0+26.2      | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability or a critical exploit (e.g., server-crashing bugs, packet exploits), please **do not** open a public GitHub Issue. Instead, use one of the private channels below to report it responsibly, preventing the issue from being exploited before a fix is released.

### How to report:
* **GitHub Private Vulnerability Reporting** 
* **Email:** You can reach me privately at: `themars.mod@proton.me`

### What to expect:
* **Response Time:** I will do my best to acknowledge your report within 48-72 hours.
* **Updates:** I will keep you updated on the progress of the analysis and the creation of a patch.
* **Credits:** If you wish, you will be credited in the changelog once the fix is made public.
